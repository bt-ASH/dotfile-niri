# 使用手册
## 在已打开的多个文件间快速切换

你**已经配置好了**这些方式：

### 1️⃣ `H` / `L` — 上下 buffer 切换（最常用）
- **`L`** → 下一个 buffer（向右切换）
- **`H`** → 上一个 buffer（向左切换）
- 已定义在 `my_mappings.lua` 第 84-89 行

用法：从 yazi 打开文件 A，再打开文件 B，按 `H` 回到文件 A，按 `L` 回到文件 B。

### 2️⃣ `,b1` ~ `,b9` — 按序号跳转
- `,b1` → 第 1 个 buffer
- `,b2` → 第 2 个 buffer
- 以此类推
- 已定义在 `_buffer_.lua`

### 3️⃣ `,ff` / `,fr` — 搜索方式
- `,ff` → snacks picker 搜索文件名（也可以只搜索已打开 buffer）
- `,fr` → 最近打开的文件列表
